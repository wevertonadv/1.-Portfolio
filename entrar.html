Tipos de profissional


Profissional Front-End

Para ser um profissional front-end, temos que ter como base as linguagem HTML, CSS e JAVASCRIPT que são as principais. HTML, é uma linguagem de marcação de texto. CSS é uma linguagem de estilo e javascript é uma linguagem de programação.

Profissional Back-End

Já para quem quer ser o profissional back-end, Precisa ter os conhecimentos nas seguintes linguagem PYTHON, JAVA e PHP.

Profissional Full-Stack

Para ser o profissional full-stack, temos que dominar a parte front-end e back-end, esse profissional entende tanto quanto front-end quanto back-end. Ele é responsavel por fazer todo o desenvolvimentoa web
